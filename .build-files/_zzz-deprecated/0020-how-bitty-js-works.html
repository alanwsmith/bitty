How It Works

The bitty-COMPONENT_VERSION API is comprised of seven data-* attributes. The timestamp demo uses three. Here's another look at its HTML:


[!- autoescape false !]
[@- data.misc_html["timestamp-demo.html"].highlighted -@]
[!- endautoescape -!]
  • The data-connect attribute makes the connection to the JavaScript module responsible for providing functionality.
  • The data-send attribute sets up the button to send the update signal when its clicked.
  • The data-receive attribute registers the div to receive the update signal when its sent.
