this.api.fn Functions

bitty looks in two places for functions to add to its API in this order

  • `const functions = {}` inside the `[@ config.script_min_file_name @]` itself
  • `window.bittyFunctions = {}` on the parent page

Any functions in those locations are added to `this.api.fn` that can be used inside signal methods. These function calls use `.bind()` so that using `this` inside them connects back to the bitty component.

NOTE: Functions will also be ingested from class modules files in Version 3.
