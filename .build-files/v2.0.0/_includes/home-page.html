Head's Up: Version 3 is pending

The API for Version 3 is the same as Version 2 with the following breaking changes:

  • this.api.useHTML() has been renamed to this.api.makeEl() to avoid confusion with this.api.getHTML() which returns a document fragment.
  • The value returned from this.api.getHTML(), .getJSON(), .getSVG(), and .getTXT() has changed. If the request is successful the result is [@ highlight_code("{ ok: PAYLOAD }", "js")|safe @]. If the request fails, the result is [@ highlight_code('{ error: { text: "MESSAGE", status: STATUS_CODE } }', "js")|safe @].
