[! filter markdown|safe !] - NOTE: This is being renamed from `useHTML` to `makeEl` in v3.x.x - Produces an HTML element from a string of text. - Performs find/replace updates from the `subs` parameter on the `content` string. - Creates a `template` and sets its `.innerHTML` to the value of the updated `content` string. - Clones the `template` and gets its `.firstChild` which is the returned as an element. NOTE: Only one element is retured. If the text snippet has more than one elemnet as sibling at the top level only the first one is returned. [! endfilter !]
