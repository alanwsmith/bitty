• Attempts to fetch a text file from the given `url`
  • If the fetch succeeds, the contents are added into the text collection at `this.api.text(key)`
  • The methods returns an object with `{ ok: true|false, error: null|payload}`
  • If the fetch succeeds, the returned value is `{ ok: true, error: null }`
  • If the fetch fails, the returned value is `{ ok: false, error: payload }`, where payload is the log connection format for the `error` level.
