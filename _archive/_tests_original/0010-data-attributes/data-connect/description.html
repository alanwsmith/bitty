• `data-connect` is an optional attribute that can be added to [@ d.bitty_tag @] tags.
  • If a `data-connect` attribute exists bitty will attempt to load the module defined by the attribute's value.
  • If a [@ d.bitty_tag @] tag doesn't have a `data-connect` attribute it will attempt to load a module class from `window.BittyClass` as a fallback.
