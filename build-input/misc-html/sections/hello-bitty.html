Bitty is a web component. Wrapping elements with it makes them reactive.

This page has examples of all of bitty's features. The first is this counter. Click it to make the number go up.

[@ data.misc_html["counter.html"].raw @]

Bitty uses a signaling approach to reactivity. data-* attributes define which signals each element sends and recieves. For example, the counter's code looks like this:

  • data-bridge connects to the functions in counter.js
  • data-c calls the _addOne function when the button is clicked (prepending the funciton name with an _ is part of bitty's naming convention)
  • data-s sends the showNum signal when the button is clicked. It fires $showNum which handles the updates (prepending $ is another part of the naming convention)
  • data-r registeres the button to receive updates when showNum signals are sent

There are more features, but that's everything you need to know to get started.
