[! filter markdown|safe !] - **data-module** connects to the script responsible for functionality. - **data-receive** registers the **div** to react to **update** signals. - **data-send** tells the **button** to send an **update** signal. #### Functionality The values from the **data-send** and **data-receive** attributes map to functions with the same name. [! endfilter !] 

  [@- highlight["modules/timestamp-demo.js"]|safe -@]
[! filter markdown|safe !] - The **update** function handles signals from elements with **data-send="update"** - Elements with **data-receive="update"** are processed by the **update** function. The two arguments passed to a **bitty-js function** are: 1. The element to update 2. The event that triggered the update The **div** element has its **innerHTML** updated with the timestamp. The event isn't used in the demo. [! endfilter !]
