Designed for Simplicity [! filter markdown|safe !] The goal is to make adding reactivity to sites and pages as simple as possible. To wit, **bitty-js** is: - A single file web component - Written in vanilla JavaScript - Powered by standard HTML attributes - Stylable via the light DOM - Designed for both progressive enhancement and API based content loading The result: - CSS selectors work like they do on regular elements. - __data-* attributes__ are the only things used in the HTML. - The elements used in the **bitty-js** functions are the same ones you'd get if you used `document.querySelector()`. - The events used in the **bitty-js** functions are the same ones you'd get if you used `document.addEventListener()`.
