## Create Docs Output

### Run jsdoc

- Update the paths in 
`pre-scripts/0010-create-jsondoc-json/generate-jsdoc-json`
if necessary to point to the current version

- Run the script from the project root (i.e.
the directory above `pre-scripts`)

### Update Docs

Go to the documentation page. It'll
show errors if there's new stuff that
needs to be documented.
