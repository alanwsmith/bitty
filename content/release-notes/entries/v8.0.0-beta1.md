### Version 8.0.0-beta-1

DATE TBD

- Added feature that loads any `<script></script>`
tags with `type="application/json"` and
an `id="ID"` attribute into a data store available
in `this.api.data(ID)`. 

- Added feature that loads any `<script></script>`
tags with `type="text/html"` and
an `id="ID"` attribute into a set of plain text
templates available in `this.api.template(ID)`. 

- Renamed `this.api.setProp(KEY, VALUE)`
to `this.api.setCSS(KEY, VALUE)` to avoid
confusion with `el.setProp(KEY, VALUE)`

- TODO: Don't do `.unwrap()` for the various
`.getHTML()` etc.. things. Only do `.unwrap_or()`
where you have to give it a fallback. That removes
a foot gun of just having `.unwrap()` that will
end up in some error stats if things don't 
load properly. 

- TODO: Added `el.setProp(KEY, VALUE)` that
adds or updates a `dataset-KEY` value if necessary.
If the value already exists and hasn't changed
no update is made.
