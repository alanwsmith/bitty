Head's Up: Version 3 is pending

The API for Version 3 is the same as Version 2 with the following breaking changes:

  1. this.api.useHTML() has been renamed to this.api.makeEl() to avoid confusion with the return value from this.api.getHTML(). Specifically, this.api.makeEl() returns and Element while this.api.getHTML() returns a document fragment.
  2. The value returned from this.api.getHTML(), .getJSON(), .getSVG(), and .getTXT() has changed. In version 2, the return value was either the payload or `undefined`. In version 3, if the request is successful the result is [@ highlight_code("{ ok: PAYLOAD }", "js")|safe @]. If the request fails, the result is [@ highlight_code('{ error: { text: "MESSAGE", status: STATUS_CODE } }', "js")|safe @].

Everything else works the same way if you want to experiment with Verison 2 without having to change code for Version 3.
