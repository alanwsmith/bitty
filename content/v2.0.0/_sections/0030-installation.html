Installation and Usage

Since bitty is a web component, there's no framework to install. Using it is done by downloading the [@ config.script_min_file_name @] file, putting it in your sites direcotry tree, and calling it with a script tag like:


[!- filter highlight_html|safe -!]

[!- endfilter !]

bitty components work by connection to a class that's responsible for providing functionality to the component. That class can be defined directly on the page. Or, it can be stored in an external file. For example, if we make a a file like:

/v2.0.0/modules/example-class.js

[!- filter highlight_javascript|safe !]
[!- include "v2.0.0/modules/example-class.js" !]
[!- endfilter !]

We can use it like this:

index.html

    [!- filter highlight_html|safe !]
    [!- include "v2.0.0/_snippets/installation/example.html" !]



    [!- endfilter !]
