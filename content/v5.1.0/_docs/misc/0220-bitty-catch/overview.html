bittyCatch(event)

Some events don't come from element and don't have an approachable way to add a data-send attribute to them (e.g. `.postMessage()`). These events are captured by an optional bittyCatch(event) method. It can be adjusted and used to trigger signals by using this.api.forward()

Note that you'll probably need to do additional filtering. For example, if bitty is set up to listen for click events, it'll trigger for every one of them. Not just for the ones associated with elements that have data-send on them
