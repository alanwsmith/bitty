UUIDs for Elements
  • Every bitty component and all child elements automatically receive a `data-bittyid` attribute with a UUID.
  • Elements added via `this.api.makeElement`, `this.api.makeHTML`, `this.api.getElement`, and `this.api.getHTML` have `data-bittyid` attributes added automatically
  • Elements added outside the `this.api...` methods (e.g. vai `document.createElement()`) _do not_ have `data-bittyid` attributes added automatically.
