Introduction

bitty is a web component. Wrapping it around other elements makes them interactive. For example, this code:

page.html

          [!- filter highlight_html|safe -!]
            [! include version_dir + "/_snippets/overview.html" !]



          [!- endfilter -!]

How It Works

bitty uses signals to send events to elements. The signals are defined with data-* attributes. They map to functions with the same name. From the example:

  • [@ highlight_code('
    ', 'html')|safe @] receives showDate signals.
  • [@ highlight_code('showDate(event, el) {}', 'js')|safe @] provides the functionality.

That's all you need to know to get started.
