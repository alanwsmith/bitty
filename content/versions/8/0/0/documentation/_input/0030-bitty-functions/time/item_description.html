[! filter markdown|safe !] - Outputs a timestamp in the format YYYY-MM-DDTHH:MM:SS for the current locale. - The hour in ``HH`` is based of a 24 hour clock (e.g. 3PM is ``15``). - If a `Date` is passed to the function the output is based off it. - If no argument is provided the function uses `new Date()` for the time. [! endfilter !]
