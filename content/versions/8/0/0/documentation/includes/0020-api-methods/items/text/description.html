  • Outputs the text from the page's storage as text after doing a find/replace for the subs.
  • The find replace items are set up in a object where they keys are the strings to find and the values are what to replace the found items with
  • Find and replace is global across the content (i.e. every instance of the `find/key` string is replaced with the `replace/value` content)

  • Replacement values can be:

    • [x] a string
    • [x] an array of strings
    • [ ] an html element
    • [ ] an array of html elements
    • [ ] a document fragment
    • [ ] an array of document fragments
    • [ ] an SVG
    • [ ] an array of SVGs

    In the cases of html elements, document fragments, and svgs, the values are turned into their text forms.
