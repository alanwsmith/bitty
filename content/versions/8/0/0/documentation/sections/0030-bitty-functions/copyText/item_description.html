[! filter markdown|safe !] - [] Copies the text from the given selector. - [] All elements that match the selector are used to populate the text (i.e. the selector is passed to querySelectorAll). - [] If multiple elements are found the string in the optional `separator` argument is used to join them. - [] If a selected element has a `.value` it's what's used for the text from the element. - [] If a selected element doesn't have a`.value`, then its `.innerHTML` is what's used for the text from the element. - [] Returns true if there was at least one element and the copy succeeded. - [] Returns false if the selector didn't match any elements. - [] Returns false if the copy failed. [! endfilter !]
