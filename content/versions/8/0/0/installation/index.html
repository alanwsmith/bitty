installation

the steps

  1. Adding bitty to a page is done by including it via a script tag. Start by downloading it:

    download bitty v8.0.0-beta4

  2. Put the file on your server. Then, call it from your pages with:

    [! filter block_highlight("html") !] [! endfilter !]

  3. Once the script tag is on your page you can use bitty tags like:

    [! filter block_highlight("html") !] [! endfilter !]

hello, bitty

using unpkg.com

Work is in progress to host bitty on unpkg.com so it can be used without having to put the file directly on your server.

using bitty with other software

bitty is designed to play nice with other software. As long as the framework/app/whatever doesn't interfere with loading scripts on a page or calling modules bitty will work just fine.

bitty will have problems if other software modifies the way things are loaded. Work-arounds may be possible. Figuring out what can work with bitty and what can't is on the roadmap.
