Overview

This page contains the documentation for bitty's entire public API. It's broken down into four categories:

  1. The `data-*` attribute that make up the HTML interface
  2. `this.api...` methods provided by bitty for use inside the JavaScipt class that's providing functionality
  3. Helper methods and properties that are added to events that come in from bitty signals (i.e. the `ev` argument in `someFeature(ev, el)` methods).
  4. Helper methods and properties that are added to the elements that are processed when a bitty signal is received (i.e. the `e.` argument in `someFeature(ev, el)` methods).

Event and Element Varaibles

bitty sends to arguemnts to the methods that process signals. The first is the event that triggered the signal. The second is the element that's currently being processed. The signture looks like this:

someMethod(ev, el) {
  // functionality goes here
}

This documentaiton uses `ev` to identify the events and `el` to identify the elements.

el.bittyId

Every element inside a <bitty-7-0> tag gets a `bittyid` property assigned to it's `dataset` with a UUID value (i.e. el.dataset.bittyid). el.bittyId is an alias to that value.

Examples
Basic Usage
HTML
<bitty-7-0 data-connect="BittyidEl">
  <div data-init="elBittyId">
    waiting
  </div>
</bitty-7-0>
JavaScript
window.BittyidEl = class {
  elBittyId(_, el) {
    el.innerHTML = el.bittyId;
  }
}

el.isSender

Boolean that returns true if the ev.sender element that sent the signal is also the receiving element.

Examples
Element IS the sender
HTML
<bitty-7-0 data-connect="IssenderEl">
  <button 
    data-send="elIsSender" 
    data-receive="elIsSender"
  >
    Example Button
  </button>
</bitty-7-0>
JavaScript
window.IssenderEl = class {
  bittyReady() {
    this.api.querySelector("button").click();
  }
  elIsSender(_, el) {
    el.innerHTML = el.isSender;
  }
}
Output
Element IS NOT the sender
HTML
<bitty-7-0 data-connect="IssenderElFalse">
  <div data-send="elIsNotSender">
    <button data-receive="elIsNotSender">
      Example Button
    </button>
  </div>
</bitty-7-0>

el.value

The standard el.value from HTML Elements.

Examples
Basic Usage
HTML
<bitty-7-0 data-connect="ValueEl">
  <input 
    type="hidden" 
    data-init="valueEl" 
    value="the quick brown fox"
  />
  <div data-receive="valueElOutput">
    waiting
  </div>
</bitty-7-0>
JavaScript
window.ValueEl = class {
  #incomingValue = null;

  valueEl(_, el) {
    this.#incomingValue = el.value;
    this.api.localTrigger("valueElOutput");
  }

  valueElOutput(_, el) {
    el.innerHTML = this.#incomingValue;
  }
}
Output
waiting

NOTE: This example uses two methods in order to show the output directly on the page. Using a single method approach would result in having to show the output in the console since updating the input value itself wouldn't look like anything changed.
