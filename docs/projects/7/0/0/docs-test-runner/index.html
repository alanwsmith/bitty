bitty project:
Documentation Test Runner For bitty

Overview

  • TODO: This is the test runner that checks the documentation for the site.

  • TODO: The output for each example in the docs has a data-expect="VALUE"attribute on it. This test running compares that value to the .innerHTMLvalue of the element and reports on it.

Example

Here's the input

<bitty-7-0 data-connect="BittyDocTestRunner ">

<div data-receive="testResults"></div>

<details open>
  <summary>Test Alfa</summary>
  <div data-expected="alfa">alfa</div>
</details>

<details>
  <summary>Test Bravo</summary>
  <div data-expected="bravo">charlie</div>
</details>

</bitty-7-0>

And here's output

Test Alfa
alfa
Test Bravo
charlie
