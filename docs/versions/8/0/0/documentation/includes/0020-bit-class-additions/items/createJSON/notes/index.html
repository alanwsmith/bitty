• Items stored in localStorage are moved under a top level `data` key in the object and stringified. (e.g. `{ "example": "alfa" }` becomes the string `{ "data": { "example": "alfa" } }` in localStorage.
  • The related bit methods work with the same top level `data` key in local storage. Those methods being:
    • `.fetchJSON(url [, options = {}])`
    • `.loadJSON(key, fallback)`
    • `.deleteJSON(key)`
    • `.saveJSON(key)`
