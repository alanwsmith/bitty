• Logs sent in through the `.trace()`, `.debug()`, `.info()`, `.warn()`, and `.error()` methods on `this.api` are passed through collection functions.
  • By default, the collection functions create an instance of the `BittyLog` class with the given payload and push it onto the array of logs available in `this.api.logs()`
  • The collection functions can be overridden to change their behaviour.
