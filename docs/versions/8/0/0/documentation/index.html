data-* attributes

Overview

  • The HTML side of the bitty API consists of these four attributes.
  • Attributes can be added to HTML elements to send and receive signals through JavaScript methods.

Details

data-connect
Description
  • `data-connect` is an optional attribute that can be added to bitty-8-0 tags.
  • If a `data-connect` attribute exists bitty will attempt to load the module defined by the attribute's value.
  • If a bitty-8-0 tag doesn't have a `data-connect` attribute it will attempt to load a module class from `window.BittyClass` as a fallback.
Examples
Load a default class from `window.BittyClass` if no `data-connect` attribute exists on a `bitty-8-0` tag.
HTML
<bitty-8-0></bitty-8-0>

<div class="test" data-receive="signal_DA553">bug</div>
JavaScript
<script>
window.BittyClass = class {
  bittyReady() {
    this.api.trigger("signal_DA553");
  }

  signal_DA553(_, el) {
    el.innerHTML = "ok";
  }
};
</script>

data-listeners
Description
Examples
Listen for a specific event type.
HTML
<bitty-8-0 data-connect="Class9CE5F"></bitty-8-0>

<div 
  class="el-9CE5F" 
  data-listeners="bittylocallistenertest"
  data-send="signal_9CE5F"
></div>

<div class="test" data-receive="signal_9CE5F">bug</div>
JavaScript
<script>
window.Class9CE5F = class {
  bittyReady() {
    const testSender = document.querySelector(".el-9CE5F");
    const testEvent = new TestEventClass9CE5F();
    testSender.dispatchEvent(testEvent);
  }

  signal_9CE5F(_, el) {
    el.innerHTML = "ok";
  }
};

// This test uses a custom event instead of
// something like mouseover to facilitate
// automated test output.
class TestEventClass9CE5F extends Event {
  constructor() {
    super("bittylocallistenertest", { bubbles: true });
  }
}
</script>
Results

      el.innerHTML = "ok";
    }
  }
};

this.api methods

Overview

Details

getStorageOr(key, alt)
Description
  • Pulls an item out of localStorage and parses it as JSON
  • If there's not an item in localStorage with the given name the alternate value is returned
Examples
Get existing value and parse it as JSON
HTML
<bitty-8-0 data-connect="Class475EF"></bitty-8-0>

<div class="test" data-receive="signal_475EF">bug</div>
JavaScript
<script>
window.Class475EF = class {
  #storageKey = "key-475EF";

  bittyReady() {
    this.api.setStorage(
      this.#storageKey,
      { status: "ok" },
    );
    this.api.trigger("signal_475EF");
  }

  signal_475EF(_, el) {
    const data = this.api.getStorageOr(
      this.#storageKey,
      { status: "bug" },
    );
    el.innerHTML = data.status;
  }
};
</script>
