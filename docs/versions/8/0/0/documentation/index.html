    • localTimestamp(date = new Date())
      Description
      • Outputs a timestamp in the format YYYY-MM-DDTHH:MM:SS for the current locale.
      • The hour in HH is based of a 24 hour clock (e.g. 3PM is 15).
      • If a Date is passed to the function the output is based off it.
      • If no argument is provided the function uses new Date() for the time.
      Examples
      • Display the current time.
        HTML
        <div data-r="signal_df095" class="test">
  waiting
</div>
<button data-s="signal_df095">
  Test df095
</button>
        JavaScript
        export const bitty = {};

export function signal_df095(ev, sender, el) {
  const timestamp = bitty.localTimestamp();
  el.innerHTML = `passed ${timestamp}`;
}
