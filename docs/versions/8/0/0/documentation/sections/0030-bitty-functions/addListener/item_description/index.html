• [] Adds event listeners for a given event and sets it up to send the corresponding signals when it occurs.
  • [] Can be used to fire signals for events that aren't associated with an element.
  • [] Can be used to fire signals that could be associated with an element (e.g. you can add a click listener that listens for any click on the page)
  • [] Signal functions are called with the three arguments: the event, undefined, undefined)
