• [x] Fetches a URL to get templates from.
  • [x] Each <template> tag at the root of the file is converted into a document fragment.
  • [x] The templates are returned as an objects.
  • [x] The id attribute on each <template> is used as the key for the template's document fragment in the object.
  • [x] <template> elements without an id are ignored.
  • [x] Options can be passed to the fetch call with the optional section argument.
