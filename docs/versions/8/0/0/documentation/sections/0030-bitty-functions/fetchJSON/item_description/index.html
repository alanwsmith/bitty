• [x] Fetches a JSON from a URL.
  • [x] Returns an object with a .value that has the JSON object if the fetch succeeded and the JSON parsed properly.
  • [] Returns an object with a .error if the fetch fails.
  • [] Returns an object with a .error if the fetch succeeds but the JSON can't be parsed.
  • [] Accepts and optional options object which is passed to the fetch call (e.g. for headers).
