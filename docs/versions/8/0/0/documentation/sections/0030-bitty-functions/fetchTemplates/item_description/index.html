• [] Fetches a URL to get templates from.
  • [] Each <template> tag at the root of the file is converted into a document fragment.
  • [] The collection of document fragments is returned as the .value of a results object.
  • [] The id attribute on each <template> is used as the key for the template's document fragment in the object.
  • [] <template> elements without an id are ignored.
  • [] The result object contains an .error instead of a .value if the URL could not be fetched.
  • [] The result object contains an .error instead of a .value if the templates could not be parsed.
  • [] Options can be passed to the fetch call with the optional section argument.
