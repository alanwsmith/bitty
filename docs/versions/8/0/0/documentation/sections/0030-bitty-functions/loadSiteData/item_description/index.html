• [] Returns a string from the site wide IndexedDB object store with the given key.
  • [] Returns an object from the site wide IndexedDB object store with the given key.
  • [] Provides for an optional object fallback to use if there's nothing in storage with the given key.
  • [] Returns undefined if there's no value for the key and there's no fallback.
  • [] Logs and error to the console and returns undefined if an error occurs.
