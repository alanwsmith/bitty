• [] Loads JSON data from localStorage if it exists and returns it.
  • [] The key is prepended with the .pathname() for the current page from the URL.
  • [] Provides for an optional object fallback to use if there's nothing in storage with the given key.
  • [] Returns undefined if the localStorage doesn't exist and there's no fallback.
  • [] Returns undefined if the value in localStorage can't be parsed as JSON.
