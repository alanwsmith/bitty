• This is a progressive enhancement approach to adding a toggle switch.
  • The page has an empty span on it with a toggle-enhancement class.
  • The module looks for any spans with the tags when it loads and replaces them with the toggle switches.
  • If the module of JavaScript fails to load there's nothing visible on the page.
  • The HTML for the switch is embedded in the module so it maintains control over it.
  • The CSS is moved to an external stylesheet so that multiple modules can use the same set of styles.
  • There's no receiving element for the signal in this snippet. The function fires a single time with no element in the third argument slot as a result.
